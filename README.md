# Setup Tor To Linux<br/>
Setup Tor Browser to Linux fast and easier.<br/>
<br/>
# Requirements<br/>
 *g++<br/>
 *Linux<br/>
 *Internet<br/>
# How to compile & run cpp file?<br/>
1) Setup g++<br/>
sudo apt-get update<br/>
sudo apt-get install g++<br/>
2) Compile it<br/>
g++ main.cpp -o setup<br/>
3) Run it<br/>
./setup<br/>
